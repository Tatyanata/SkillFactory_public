команды git загрузить новый файл в уже имеющийся репо
git add (имя файла c форматом данных)
git commit -m "сообщение об изменениях об изменениях"
git push

удалить файл из репы, из системы контроля версий
git rm (имя файла c форматом данных)
git status
git commit -m "сообщение о том, что делаем"
git push

склонировать удаленный репо к себе на комп 
git clone (копировать https адрес в гите с пом.зел.кнопки)
git add . 
git commit -m "написать сообщение"
git push
